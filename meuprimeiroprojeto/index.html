Introdução ao HTML


O que são Tags?

Em HTML, uma tag é um código usado para marcar e formatar elementos em uma página da web. As tags são elementos de marcação que dizem ao navegador como exibir o conteúdo. Elas funcionam como "etiquetas" que indicam o início e o fim de diferentes tipos de conteúdo, como títulos, parágrafos, imagens, etc.

O que são Atributos?

Em HTML, atributos são palavras especiais usadas dentro de tags para adicionar informações ou definir o comportamento de um elemento HTML. Eles fornecem detalhes adicionais sobre como um elemento deve ser exibido ou como ele deve funcionar.

URLs

A tag "a"define um hiperlink. O atributo "href"especifica a URL da página para a qual o link direciona

A tag "img" é usada para incorporar uma imagem em uma página HTML.O atributo "src"especifica o caminho para a imagem a ser exibida

O atributo "alt" da tag "img" especifica um texto alternativo para uma imagem, caso ela não possa ser exibida por algum motivo. Isso pode ocorrer devido a uma conexão lenta, a um erro no atributo "src" ou se o usuário utilizar um leitor de tela.

URL absoluta:inclui todo o endereço da página (protocolo, domínio e caminho)

URL relativa: especifica apenas o caminho para o recurso em relação à página atual
